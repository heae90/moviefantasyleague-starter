# CineDraft starter website

This is a beginner-friendly Next.js starter for your CineDraft website.

## The easiest deployment path

1. Download this ZIP and unzip it on your computer.
2. Create a free GitHub account if you do not already have one.
3. Create a new GitHub repository called `cinedraft-starter`.
4. Upload all of these files into that repo.
5. In Vercel, click **Add New Project**.
6. Import your GitHub repository.
7. Click **Deploy**.

Vercel officially supports deploying from a Git repository in the dashboard, and it also supports deploying a local project with the CLI.

## After deployment

Your live site will get a URL like:
`https://cinedraft-starter.vercel.app`

## Files you will edit first

- `app/page.tsx` → page content
- `app/globals.css` → colors and styling

## If you want local preview later

Install Node.js, then in the project folder run:

```bash
npm install
npm run dev
```

Then open:
`http://localhost:3000`
